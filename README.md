# 🤟 SignBridge AI – Indian Sign Language Translation System

## 📖 Overview

SignBridge AI is an AI-powered Indian Sign Language (ISL) Translation System designed to enable seamless communication between hearing-impaired and non-signing individuals. The platform leverages Computer Vision, Deep Learning, and Natural Language Processing (NLP) to recognize Indian Sign Language gestures in real time and translate them into text and speech. It also converts text and voice inputs into animated ISL gestures using a 3D avatar for two-way communication.

---

## 🎯 Objective

To bridge communication gaps by providing a real-time, accurate, and accessible Indian Sign Language translation platform that promotes inclusivity and equal communication opportunities for the deaf and hard-of-hearing community.

## 🔄 Workflow

1. User performs ISL gestures in front of a camera.
2. Camera captures hand, face, and body movements.
3. Computer Vision extracts key landmarks.
4. Deep Learning model identifies the sign.
5. NLP processes and refines the output.
6. System converts the sign into text and speech.
7. Text or voice input is converted into ISL.
8. 3D avatar displays corresponding sign language animations.
